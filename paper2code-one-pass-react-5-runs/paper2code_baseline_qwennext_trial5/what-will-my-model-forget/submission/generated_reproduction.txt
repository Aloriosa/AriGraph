# Forecasting Forgotten Examples in Language Model Refinement (Reproduction)

This repository contains a lightweight, fully reproducible implementation of the key ideas from the paper *“What Will My Model Forget? Forecasting Forgotten Examples in Language Model Refinement”*.
The goal is to show, with a small synthetic toy dataset, how one can:

1. Fine‑tune a pretrained seq2seq model on a single *error* example.
2. Measure which *upstream* training examples have become *forgotten*.
3. Train a simple forecasting model that predicts forgetting based on
   learned sentence embeddings.
4. Use the forecast to replay the most likely forgotten examples and
   observe a reduction in catastrophic forgetting.

The implementation is intentionally lightweight – it uses the
`t5-small` model (≈ 60 M parameters) and a tiny synthetic dataset
(≈ 200 sentence pairs).  It can be run on CPU or GPU, and the
`reproduce.sh` script will automatically install all necessary
dependencies.
