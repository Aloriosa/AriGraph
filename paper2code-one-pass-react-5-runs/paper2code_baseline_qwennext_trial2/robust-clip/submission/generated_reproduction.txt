# Reproducing Robust CLIP (FARE) on CIFAR‑10

This repository contains a minimal, self‑contained reproduction of the main ideas from  
“Robust CLIP: Unsupervised Adversarial Fine‑Tuning of Vision Embeddings for Robust Large Vision‑Language Models”.

We download the pre‑trained CLIP model from Hugging Face, fine‑tune it on CIFAR‑10 with a simple
adversarial training scheme (PGD), and evaluate clean and adversarial accuracy.
The workflow is fully automated through `reproduce.sh`.

> **NOTE**: The full paper evaluates LVLMs on COCO, VQA, etc.  
> Implementing the entire pipeline would require several GPU hours and >1 GB of data.  
> This repository focuses on the *core* concept of unsupervised adversarial fine‑tuning
> (FARE) and demonstrates it on a small, publicly available dataset.
