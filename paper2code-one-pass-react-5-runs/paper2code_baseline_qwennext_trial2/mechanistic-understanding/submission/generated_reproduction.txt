# Reproduction of “A Mechanistic Understanding of Alignment Algorithms:
# A Case Study on DPO and Toxicity”

This repository contains a minimal, fully‑automatic reproduction of the main
experimental pipeline described in the paper.  
The reproduction demonstrates:

1. **Training a toxicity probe** on a small subset of the Jigsaw toxicity
   dataset using GPT‑2‑medium.
2. **Training a Direct Preference Optimization (DPO)** model on a toy
   pairwise dataset (positive = greedy continuation, negative = random
   continuation) derived from Wikitext‑2.
3. **Evaluating toxicity** before and after DPO by counting a small set of
   profanity words in generated continuations.

> **Note** – This is a *toy* version of the full experiments.  
> The original paper trains on thousands of pairs and uses a sophisticated
> PPLM‑generated negative dataset.  The code below keeps the same
> overall workflow but reduces data size and training time so that the
> reproduction runs in < 30 minutes on a single GPU.
