# Refined Coreset Selection: Towards Minimal Coreset Size under Model Performance Constraints

This repository contains the source code to reproduce the results presented in the paper "Refined Coreset Selection: Towards Minimal Coreset Size under Model Performance Constraints" (ICML 2024).

## Overview

The paper introduces the problem of **Refined Coreset Selection (RCS)**, which aims to find the smallest possible coreset that maintains comparable model performance to training on the full dataset. The paper proposes a novel method, **Lexicographic Bilevel Coreset Selection (LBCS)**, which formulates RCS as a cardinality-constrained bilevel optimization problem with lexicographic preferences over two objectives: (1) maximizing model performance and (2) minimizing coreset size.

## Repository Structure

- `reproduce.sh`: The reproduction script to execute the code.
- `leco.py`: The main Python script implementing the LBCS algorithm for RCS.
- `output/`: Directory to store the output files generated by the reproduction script.
