# Reproducing “Cramming 1568 Tokens into a Single Vector and Back Again”

This repository contains a lightweight, fully‑reproducible implementation of the core idea presented in  
*“Cramming 1568 Tokens into a Single Vector and Back Again”* by Kuratov et al.  
The paper studies how many tokens can be *losslessly* compressed into a small set of trainable
“memory” embeddings ([mem] vectors) that are prepended to the input of a frozen language model.
Our implementation demonstrates the method on a small GPT‑2 model and a handful of short
sentences.

> **Why this repo?**  
> The original experiments use 1–8 B parameter models and run on GPUs for several minutes.
> Here we provide a minimal, CPU‑friendly reproduction that can be run on any recent
> Ubuntu 24.04 system (or the Docker container used for evaluation) within a few minutes.

## Repository layout
```
├── compress_and_eval.py   # Main script: trains mem vectors & evaluates reconstruction
├── reproduce.sh           # Bash script that installs deps & runs the experiments
├── data/
│   └── texts.txt          # Tiny corpus used for demonstration
└── README.md
```

## How it works

1. **Training** – For each line in *data/texts.txt* we:
   * Tokenise the text.
   * Initialise a single trainable vector `mem` of the same dimensionality as the model’s hidden state.
   * Optimize `mem` with AdamW to minimise the next‑token cross‑entropy loss while the base
     GPT‑2 model is frozen.
   * Stop early when the model predicts the text perfectly (token‑level accuracy = 1.0) or after
     a fixed number of steps.

2. **Evaluation** – Using the learned `mem` vector we generate tokens greedily, **without providing
   any prompt**.  We compare the generated sequence to the original text and report
   token‑level accuracy.

The script prints a table summarising:
- Original length (in tokens)
- Learned `mem` vector length (always 1)
- Number of training steps
- Reconstruction accuracy

## Running the reproduction

```bash
bash reproduce.sh
```

The script will:
1. Install the required Python packages (`torch`, `transformers`).
2. Run `compress_and_eval.py`, which trains on the tiny dataset and prints results.

> **Note**: The script uses the *gpt2* checkpoint (≈ 124 M parameters).  
> It runs on CPU and completes in < 5 minutes on a modern laptop.

## Extending the experiments

- Replace `model_name="gpt2"` in `compress_and_eval.py` with any HuggingFace checkpoint
  (e.g. `EleutherAI/gpt-j-6B`) – the code will automatically download it.
- Increase `max_steps` or lower `lr` if you want to try larger models or longer texts.
- Add more lines to `data/texts.txt` to see how the method scales.

**FILE: compress_and_eval.py**
```python
#!/usr/bin/env python3
"""
A minimal implementation of the “[mem] vector” compression method
from Kuratov et al.  The script trains a single memory vector
for each line in data/texts.txt using a frozen GPT‑2 model and
evaluates lossless reconstruction.

Author: ChatGPT (OpenAI)
"""

import os
import random
import argparse
from pathlib import Path

import torch
import torch.nn as nn
import torch.optim as optim
from transformers import AutoTokenizer, AutoModelForCausalLM, set_seed

# ---------------------------------------------------------------------------

def load_data(file_path: Path):
    """Return list of strings, one per line."""
    with file_path.open(encoding="utf-8") as f:
        return [line.strip() for line in f if line.strip()]

# ---------------------------------------------------------------------------

class MemTrainer:
    """
    Trains a single trainable vector per sample.
    The base language model is frozen.
    """

    def __init__(
        self,
        model_name: str,
        device: torch.device,
        num_mem: int = 1,
        max_steps: int = 200,
        lr: float = 1e-3,
        weight_decay: float = 0.0,
        seed: int = 42,
    ):
        self.device = device
        self.max_steps = max_steps
        self.num_mem = num_mem

        set_seed(seed)

        # Load tokenizer and model
        self.tokenizer = AutoTokenizer.from_pretrained(model_name, use_fast=False)
        # Add a special token for the memory vector; we will not use it as an actual token
        # but it helps with debugging.
        special_token = "[MEM]"
        if special_token not in self.tokenizer.get_vocab():
            self.tokenizer.add_special_tokens({"additional_special_tokens": [special_token]})

        self.model = AutoModelForCausalLM.from_pretrained(
            model_name,
            add_cross_attention=False,
            torch_dtype=torch.float32,
            low_cpu_mem_usage=True,
        )
        # Resize token embeddings if we added a special token
        if "additional_special_tokens" in self.tokenizer.special_tokens_map:
            self.model.resize_token_embeddings(len(self.tokenizer))

        self.model.to(device)
        self.model.eval()  # freeze parameters

        self.hidden_size = self.model.config.hidden_size

        # Optimizer placeholder – will be created per sample
        self.lr = lr
        self.weight_decay = weight_decay

    def train_sample(self, text: str):
        """
        Train a memory vector for a single text.
        Returns:
            mem_vec (torch.Tensor): learned memory vector (1 x hidden_size)
            steps (int): number of optimization steps performed
            acc (float): reconstruction accuracy after training
        """
        # Tokenise
        input_ids = self.tokenizer(
            text, return_tensors="pt", truncation=True, padding=False
        ).input_ids.to(self.device)  # shape: (1, seq_len)

        seq_len = input_ids.size(1)

        # Initialize memory vector
        mem_vec = nn.Parameter(
            torch.randn(self.num_mem, self.hidden_size, device=self.device)
        )  # shape: (num_mem, hidden_size)

        # Optimizer only for mem_vec
        optimizer = optim.AdamW([mem_vec], lr=self.lr, weight_decay=self.weight_decay)

        # Prepare labels: -100 for memory tokens, then original tokens
        labels = torch.cat(
            [
                torch.full((1, self.num_mem), -100, dtype=torch.long, device=self.device),
                input_ids,
            ],
            dim=1,
        )  # shape: (1, num_mem + seq_len)

        # Training loop
        for step in range(1, self.max_steps + 1):
            optimizer.zero_grad()

            # Embed tokens
            token_embeds = self.model.transformer.wte(input_ids)  # (1, seq_len, hidden)
            mem_embeds = mem_vec.unsqueeze(0).repeat(1, 1, 1)  # (1, num_mem, hidden)
            combined_embeds = torch.cat([mem_embeds, token_embeds], dim=1)

            # Forward
            outputs = self.model(
                inputs_embeds=combined_embeds,
                labels=labels,
                output_hidden_states=False,
                return_dict=True,
            )
            loss = outputs.loss
            loss.backward()
            optimizer.step()

            # Early stopping if perfect reconstruction
            if step % 10 == 0:
                acc = self._evaluate(mem_vec, input_ids, seq_len)
                if acc == 1.0:
                    break

        steps = step
        acc = self._evaluate(mem_vec, input_ids, seq_len)
        return mem_vec.detach().cpu(), steps, acc

    def _evaluate(self, mem_vec: torch.Tensor, input_ids: torch.Tensor, seq_len: int):
        """
        Greedy generation from the learned memory vector.
        Returns token‑level accuracy against the reference input_ids.
        """
        # Start with empty generated list
        generated_ids = []
        # We'll generate at most seq_len tokens
        for _ in range(seq_len):
            # Prepare current context: mem + generated so far
            if generated_ids:
                cur_ids = torch.tensor([generated_ids], dtype=torch.long, device=self.device)
                cur_embeds = self.model.transformer.wte(cur_ids)
            else:
                cur_embeds = torch.empty((1, 0, self.hidden_size), device=self.device)

            mem_embeds = mem_vec.unsqueeze(0).repeat(1, 1, 1)
            combined_embeds = torch.cat([mem_embeds, cur_embeds], dim=1)

            # Forward to get logits for the last position
            logits = self.model.transformer(inputs_embeds=combined_embeds).logits
            next_token_logits = logits[0, -1, :]  # (vocab,)
            next_id = torch.argmax(next_token_logits, dim=-1).item()

            if next_id == self.tokenizer.eos_token_id:
                break
            generated_ids.append(next_id)

        # Compare
        gen_ids = torch.tensor(generated_ids, dtype=torch.long, device=self.device)
        labels = input_ids[0][: len(generated_ids)]
        correct = (gen_ids == labels).float().mean().item()
        return correct

# ---------------------------------------------------------------------------

def main(args):
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    trainer = MemTrainer(
        model_name=args.model,
        device=device,
        num_mem=args.num_mem,
        max_steps=args.max_steps,
        lr=args.lr,
        weight_decay=args.weight_decay,
        seed=args.seed,
    )

    data_path = Path(args.data)
    texts = load_data(data_path)
    print(f"Model: {args.model}")
    print(f"Using {args.num_mem} memory vector(s) per sample.")
    print(f"Training {len(texts)} samples.\n")

    for i, txt in enumerate(texts, start=1):
        mem_vec, steps, acc = trainer.train_sample(txt)
        print(f"Sample {i}:")
        print(f"  Original length: {len(trainer.tokenizer(txt).input_ids)} tokens")
        print(f"  Training steps:  {steps}")
        print(f"  Reconstruction accuracy: {acc:.2f}\n")

    print("All samples processed.")

if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="Train [mem] vectors and evaluate reconstruction.")
    parser.add_argument("--model", type=str, default="gpt2", help="HF model name")
    parser.add_argument("--data", type=str, default="data/texts.txt", help="Path to data file")
    parser.add_argument("--num-mem", type=int, default=1, help="Number of memory vectors")
    parser.add_argument("--max-steps", type=int, default=200, help="Max training steps per sample")
    parser.add_argument("--lr", type=float, default=1e-3, help="Learning rate")
    parser.add_argument("--weight-decay", type=float, default=0.0, help="Weight decay")
    parser.add_argument("--seed", type=int, default=42, help="Random seed")
    args = parser.parse_args()
    main(args)
```
