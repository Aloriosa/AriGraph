# Reproduction: Batch and Match (BaM) for Black-Box Variational Inference

This repository reproduces the results from the paper "Batch and match: black-box variational inference with a score-based divergence" by Cai et al. (2024).

## Overview

The paper introduces Batch and Match (BaM), a novel algorithm for black-box variational inference (BBVI) that uses a score-based divergence instead of the traditional evidence lower bound (ELBO).

BaM alternates between:
1. **Batch step**: Draw samples from the current variational distribution
2. **Match step**: Update the variational parameters to match the scores of the target distribution

The key contributions are:
- A new score-based divergence for BBVI
- Closed-form proximal updates for Gaussian variational families
- Provable exponential convergence guarantees for Gaussian targets
- Empirical superiority over ADVI and Gaussian Score Matching (GSM)

## Implementation Details

The implementation follows the exact mathematical formulations from the paper:

1. **Score-based divergence**: 
   `D(q; p) = E_q[||∇_z log(q(z)/p(z))||^2_Cov(q)`

2. **Batch step**: Sample from current variational distribution `N(μ_t, Σ_t)` and compute batch statistics (means and covariances of samples and their scores)

3. **Match step**: Update variational parameters with closed-form updates derived from minimizing the score-based divergence with KL regularization

4. **Closed-form updates**: 
   - Update covariance: `Σ_{t+1} = 2V(I + (I + 4UV)^(1/2))^(-1)`
   - Update mean: `μ_{t+1} = (1/(1+λ_t))μ_t + (λ_t/(1+λ_t))(Σ_{t+1}g̅ + z̅)`

The implementation uses JAX for automatic differentiation and GPU acceleration.

#!/usr/bin/env python3
"""
Batch and Match (BaM) for Black-Box Variational Inference

This implementation reproduces the Batch and Match (BaM) algorithm from the paper:
"Batch and match: black-box variational inference with a score-based divergence"
by Cai et al. (2024)

The algorithm alternates between:
1. Batch step: Draw samples from current variational distribution
2. Match step: Update variational parameters to match the scores of the target distribution
"""

import numpy as np
import jax
import jax.numpy as jnp
from jax import random
import time
import os
from typing import Tuple, Callable
import matplotlib.pyplot as plt
from scipy.stats import multivariate_normal
from scipy.optimize import minimize

# Set random seeds for reproducibility
np.random.seed(42)
key = random.PRNGKey(42)

class BatchAndMatch:
    """
    Implementation of Batch and Match (BaM) algorithm for variational inference
    using a score-based divergence.
    """
    
    def __init__(self, target_log_density: Callable, initial_mean: np.ndarray, 
                 initial_covariance: np.ndarray, batch_size: int = 10, 
                 regularization: float = 1.0, max_iterations: int = 100):
        """
        Initialize the BaM algorithm.
        
        Args:
            target_log_density: Function that computes log p(z) (unnormalized target density)
            initial_mean: Initial mean vector for variational distribution
            initial_covariance: Initial covariance matrix for variational distribution
            batch_size: Size of batch for sampling
            regularization: Regularization parameter lambda
            max_iterations: Maximum number of iterations
        """
        self.target_log_density = target_log_density
        self.mu = jnp.array(initial_mean, dtype=jnp.float32)
        self.Sigma = jnp.array(initial_covariance, dtype=jnp.float32)
        self.B = batch_size
        self.lambda_t = regularization
        self.max_iters = max_iterations
        
        # Initialize variational parameters
        self.mu_t = self.mu
        self.Sigma_t = self.Sigma
        
        # Store history for plotting
        self.history = {
            'means': [],
            'covariances': [],
            'divergences': [],
            'iterations': []
        }
    
    def score_function(self, z: jnp.ndarray) -> jnp.ndarray:
        """
        Compute the score function (gradient of log target density)
        Using JAX's automatic differentiation
        """
        # Use JAX's grad function to compute gradient of log density
        score_fn = jax.grad(self.target_log_density)
        return score_fn(z)
    
    def batch_step(self, key: jax.random.PRNGKey) -> Tuple[jnp.ndarray, jnp.ndarray, jnp.ndarray, jnp.ndarray]:
        """
        Batch step: Draw samples from current variational distribution
        and compute batch statistics (means and covariances)
        
        Returns:
            z_batch: Batch of samples
            g_batch: Batch of scores
            z_mean: Mean of samples
            g_mean: Mean of scores
            z_cov: Covariance of samples
            g_cov: Covariance of scores
        """
        # Sample from current variational distribution
        # z ~ N(μ_t, Σ_t)
        z_batch = random.multivariate_normal(key, self.mu_t, self.Sigma_t, shape=(self.B,))
        
        # Compute scores: g = ∇_z log p(z)
        g_batch = jax.vmap(self.score_function)(z_batch)
        
        # Compute batch statistics
        z_mean = jnp.mean(z_batch, axis=0)
        g_mean = jnp.mean(g_batch, axis=0)
        
        # Covariances
        z_cov = jnp.cov(z_batch.T)
        g_cov = jnp.cov(g_batch.T)
        
        return z_batch, g_batch, z_mean, g_mean, z_cov, g_cov
    
    def match_step(self, z_mean: jnp.ndarray, g_mean: jnp.ndarray, 
                   z_cov: jnp.ndarray, g_cov: jnp.ndarray) -> Tuple[jnp.ndarray, jnp.ndarray]:
        """
        Match step: Update variational parameters to match the scores of the target distribution
        using closed-form updates
        
        Updates:
        - Σ_{t+1} = 2V(I + (I + 4UV)^(1/2))^(-1)
        - μ_{t+1} = (1/(1+λ_t))μ_t + (λ_t/(1+λ_t))(Σ_{t+1}g̅ + z̅)
        
        Args:
            z_mean: Mean of samples
            g_mean: Mean of scores
            z_cov: Covariance of samples
            g_cov: Covariance of scores
        
        Returns:
            new_mu: Updated mean
            new_sigma: Updated covariance
        """
        # Compute matrices U and V for the quadratic matrix equation
        # U = λ_t * g_cov + (λ_t/(1+λ_t)) * g_mean * g_mean.T
        # V = Σ_t + λ_t * z_cov + (λ_t/(1+λ_t)) * (μ_t - z_mean) * (μ_t - z_mean).T
        
        # U matrix
        U = self.lambda_t * g_cov + (self.lambda_t / (1 + self.lambda_t)) * jnp.outer(g_mean, g_mean)
        
        # V matrix
        V = self.Sigma_t + self.lambda_t * z_cov + (self.lambda_t / (1 + self.lambda_t)) * jnp.outer(self.mu_t - z_mean, self.mu_t - z_mean)
        
        # Solve the quadratic matrix equation: Σ_{t+1} U Σ_{t+1} + Σ_{t+1} = V
        # Solution: Σ_{t+1} = 2V(I + (I + 4UV)^(1/2))^(-1)
        
        # Compute UV product
        UV = jnp.matmul(U, V)
        
        # Compute I + 4UV
        I_plus_4UV = jnp.eye(U.shape[0]) + 4 * UV
        
        # Compute square root of (I + 4UV)
        # Use eigenvalue decomposition to compute matrix square root
        eigenvals, eigenvecs = jnp.linalg.eigh(I_plus_4UV)
        sqrt_eigenvals = jnp.sqrt(eigenvals)
        sqrt_I_plus_4UV = jnp.dot(eigenvecs, jnp.dot(jnp.diag(sqrt_eigenvals), eigenvecs.T))
        
        # Compute I + sqrt(I + 4UV)
        I_plus_sqrt_I_plus_4UV = jnp.eye(U.shape[0]) + sqrt_I_plus_4UV
        
        # Compute inverse of (I + sqrt(I + 4UV))
        inv_I_plus_sqrt_I_plus_4UV = jnp.linalg.inv(I_plus_sqrt_sqrt_I_plus_4UV)
        
        # Compute new covariance
        new_Sigma = 2 * jnp.dot(V, inv_I_plus_sqrt_sqrt_I_plus_4UV)
        
        # Update mean
        new_mu = (1 / (1 + self.lambda_t)) * self.mu_t + (self.lambda_t / (1 + self.lambda_t)) * (jnp.dot(new_Sigma, g_mean) + z_mean)
        
        return new_mu, new_Sigma
    
    def optimize(self, key: jax.random.PRNGKey) -> Tuple[jnp.ndarray, jnp.ndarray]:
        """
        Run the BaM algorithm
        """
        start_time = time.time()
        
        for t in range(self.max_iters):
            # Batch step
            z_batch, g_batch, z_mean, g_mean, z_cov, g_cov = self.batch_step(key)
            
            # Match step
            new_mu, new_Sigma = self.match_step(z_mean, g_mean, z_cov, g_cov)
            
            # Update variational parameters
            self.mu_t = new_mu
            self.Sigma_t = new_Sigma
            
            # Compute divergence (for logging)
            divergence = self.compute_divergence()
            self.history['means'].append(self.mu_t)
            self.history['covariances'].append(self.Sigma_t)
            self.history['divergences'].append(divergence)
            self.history['iterations'].append(t)
            
            # Print progress
            if t % 10 == 0:
                print(f"Iteration {t}, Divergence: {divergence:.6f}")
        
        end_time = time.time()
        print(f"Optimization completed in {end_time - start_time:.2f} seconds")
        
        return self.mu_t, self.Sigma_t
