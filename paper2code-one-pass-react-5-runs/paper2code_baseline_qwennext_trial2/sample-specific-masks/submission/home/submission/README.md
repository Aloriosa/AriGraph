# Reproduction Repository for "Sample-specific Masks for Visual Reprogramming-based Prompting"

## Overview

This repository contains the complete implementation to reproduce the results from the paper "Sample-specific Masks for Visual Reprogramming-based Prompting" by Cai et al. (2024).

The paper proposes Sample-specific Multi-channel Masks (SMM), a novel framework for Visual Reprogramming (VR) that replaces the traditional shared mask with sample-specific three-channel masks generated by a lightweight ConvNet. SMM demonstrates improved performance over existing VR methods by adapting the mask for each individual sample, reducing approximation error.
