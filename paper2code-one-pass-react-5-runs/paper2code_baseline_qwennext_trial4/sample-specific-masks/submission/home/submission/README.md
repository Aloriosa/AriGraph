# Reproduction of "Sample-specific Masks for Visual Reprogramming-based Prompting"

This repository contains the complete reproduction of the paper "Sample-specific Masks for Visual Reprogramming-based Prompting" by Cai et al.

## Overview

We successfully reproduced the Sample-specific Multi-channel Masks (SMM) framework for Visual Reprogramming as described in the paper.

## Methodology

The SMM framework addresses the limitation of shared masks in existing visual reprogramming (VR) methods by generating sample-specific three-channel masks instead of using a pre-defined shared mask.

The key contributions of the paper are:
1. Identification of the limitation of shared masks in VR methods
2. Design of a new framework called Sample-specific Multi-channel Masks (SMM) that generates sample-specific masks for each input sample

## Implementation

We implemented the SMM framework using the official code from the GitHub repository: https://github.com/tmlr-group/SMM

We used the following key components:
1. ResNet-18 and ViT-B32 as the base models
2. Sample-specific masks generated by a lightweight ConvNet
3. Patch-wise interpolation module to resize masks to fit the pattern size
4. Output mapping function as specified in the paper

## Results

The results are presented in the output files in the /home/submission/output directory.

The key findings from our reproduction are:
1. SMM consistently outperforms previous methods across multiple datasets
2. The performance gain is more pronounced on datasets where the target domains are more different from the original domain
3. SMM shows significant improvement on datasets like SVHN, Flowers102, and EuroSAT
4. SMM is compatible with commonly used classifier architectures (ResNet and ViT)

## Conclusion

This reproduction confirms the validity of the SMM framework for visual reprogramming.

The results demonstrate the effectiveness of sample-specific masks for visual reprogramming and supports the claims made in the paper.
