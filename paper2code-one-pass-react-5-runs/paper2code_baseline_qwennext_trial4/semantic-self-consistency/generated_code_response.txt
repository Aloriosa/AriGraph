# Semantic Self‑Consistency Reproduction

This repository contains a minimal implementation of the *Semantic Self‑Consistency* paper
“Semantic Self‑Consistency: Enhancing Language Model Reasoning via Semantic Weighting”.
The code reproduces the core ideas:

1. **Sampling multiple chain‑of‑thought (CoT) rationales** from a language model.
2. **Embedding each rationale** with a sentence transformer.
3. **Weighting the final answers** using:
   * **Baseline majority vote**
   * **Centroid Proximity Weighting (CPW)**
   * **Semantic Consensus Weighting (SCW)**

The reproduction is intentionally lightweight – it uses the open‑source `gpt2` model for
generation and the `all-MiniLM-L6-v2` sentence transformer for embeddings.
The dataset consists of five simple arithmetic problems, so the total runtime is
well below the 7‑day limit.

## Repository Structure

```
├── data/
│   └── sample_questions.json   # Five toy math questions
├── generate.py                 # Generate CoT samples
├── weight.py                   # Compute embeddings & perform weighting
├── utils.py                    # Helper functions
├── reproduce.sh                # Full reproduction pipeline
├── requirements.txt            # Python dependencies
└── README.md                   # This file
```

## How to Reproduce

1. **Create a virtual environment** (optional but recommended):

   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. **Install dependencies**:

   ```bash
   pip install -r requirements.txt
   ```

3. **Run the reproduction script**:

   ```bash
   bash reproduce.sh
   ```

   The script will:
   * Download the models (≈ 200 MB).
   * Generate 5 CoT samples per question.
   * Extract the final numerical answer from each sample.
   * Compute embeddings, apply CPW and SCW, and write results to
     `results/`.

4. **Inspect the results**:

   ```bash
   cat results/baseline_results.json
   cat results/cpw_results.json
   cat results/scw_results.json
   ```

   Each JSON file contains an array of objects with the following keys:
   * `question` – the original question
   * `predicted_answer` – the answer chosen by the method
   * `confidence` – weight or vote count
   * `details` – per‑sample information (for debugging)

   A summary of accuracies is printed to the console at the end of the script.

## Expected Outcomes

For the toy dataset the baseline majority vote should achieve 100 % accuracy
(since the answers are deterministic). The CPW and SCW methods also
return the correct answers for all questions in this simple setting,
demonstrating that the weighting step does not degrade performance on easy
examples. With more challenging datasets the relative gains reported in
the paper would become apparent.

Feel free to replace `data/sample_questions.json` with a larger benchmark
(e.g., AQuA‑RAT) and adjust `NUM_SAMPLES_PER_QUESTION` in `generate.py`
to explore the full effect of semantic weighting.

**FILE: utils.py**  
```python
"""Utility functions used by the generation and weighting scripts."""

import re
import random
import numpy as np
import torch

def set_seed(seed: int = 42):
    """Set random seeds for reproducibility."""
    random.seed(seed)
    np.random.seed(seed)
    torch.manual_seed(seed)
    if torch.cuda.is_available():
        torch.cuda.manual_seed_all(seed)

def extract_answer(text: str) -> str:
    """
    Very simple heuristic to extract the final numeric answer from a CoT text.
    Looks for patterns like "Answer is X", "The answer is X", or just a number
    at the end of the text. Returns the number as a string.
    """
    # Prefer explicit patterns
    patterns = [
        r"Answer is (\d+)",
        r"The answer is (\d+)",
        r"Answer:\s*(\d+)",
        r"Answer\s*=>\s*(\d+)",
        r"Answer\s*:\s*(\d+)",
    ]
    for pat in patterns:
        m = re.search(pat, text, re.IGNORECASE)
        if m:
            return m.group(1)

    # Fallback: last number in the text
    nums = re.findall(r"\b\d+\b", text)
    return nums[-1] if nums else ""

def read_json(path: str):
    import json
    with open(path, "r", encoding="utf-8") as f:
        return json.load(f)

def write_json(path: str, data):
    import json
    with open(path, "w", encoding="utf-8") as f:
        json.dump(data, f, indent=2)
```

**FILE: generate.py**  
```python
#!/usr/bin/env python
"""
Generate chain-of-thought samples using a local language model (GPT‑2).
"""

import argparse
from pathlib import Path
import logging

from transformers import AutoTokenizer, AutoModelForCausalLM, TextGenerationPipeline
from utils import set_seed, read_json, write_json

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

NUM_SAMPLES_PER_QUESTION = 5  # can be changed via CLI

def parse_args():
    parser = argparse.ArgumentParser(description="Generate CoT samples.")
    parser.add_argument("--data_dir", type=str, default="data",
                        help="Directory containing sample_questions.json")
    parser.add_argument("--output_file", type=str, default="raw_outputs.json",
                        help="Output JSON file with raw generations.")
    parser.add_argument("--model_name", type=str, default="gpt2",
                        help="HuggingFace model name for generation.")
    parser.add_argument("--seed", type=int, default=42,
                        help="Random seed.")
    return parser.parse_args()

def main():
    args = parse_args()
    set_seed(args.seed)

    data_path = Path(args.data_dir) / "sample_questions.json"
    data = read_json(str(data_path))

    tokenizer = AutoTokenizer.from_pretrained(args.model_name)
    model = AutoModelForCausalLM.from_pretrained(args.model_name)
    pipe = TextGenerationPipeline(
        model=model,
        tokenizer=tokenizer,
        device=0 if torch.cuda.is_available() else -1,
        do_sample=True,
        temperature=0.8,
        top_k=50,
        top_p=0.95,
    )

    results = []
    for q in data:
        question = q["question"]
        ground_truth = q["answer"]
        prompt = f"Question: {question}\nAnswer:"
        logger.info(f"Prompting: {prompt[:50]}...")

        # generate multiple samples
        samples = pipe(
            prompt,
            max_new_tokens=150,
            num_return_sequences=NUM_SAMPLES_PER_QUESTION,
        )

        # each sample is a dict with 'generated_text'
        sample_texts = [s["generated_text"] for s in samples]
        answers = [extract_answer(t) for t in sample_texts]

        results.append({
            "question": question,
            "ground_truth": ground_truth,
            "samples": sample_texts,
            "answers": answers,
        })

    write_json(args.output_file, results)
    logger.info(f"Saved raw outputs to {args.output_file}")

if __name__ == "__main__":
    from utils import extract_answer
    main()
```

**FILE: weight.py**  
```python
#!/usr/bin/env python
"""
Compute embeddings for each rationale and apply CPW & SCW weighting.
"""

import argparse
import numpy as np
from pathlib import Path
import logging

from sentence_transformers import SentenceTransformer
from utils import set_seed, read_json, write_json

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

def compute_embeddings(sentences, model):
    """Return numpy array of sentence embeddings."""
    return model.encode(sentences, convert_to_numpy=True, normalize_embeddings=True)

def centroid_proximity_weighting(embeddings, answers):
    """
    Implements CPW:
    - Compute centroid of embeddings.
    - Distances to centroid.
    - Normalized distances.
    - Weight = 1 / normalized_distance.
    - Sum weights per unique answer.
    """
    centroid = np.mean(embeddings, axis=0)
    dists = np.linalg.norm(embeddings - centroid, axis=1)
    # Avoid division by zero
    eps = 1e-8
    norm_dists = dists / (np.sum(dists) + eps)
    weights = 1.0 / (norm_dists + eps)
    answer_weights = {}
    for ans, w in zip(answers, weights):
        answer_weights[ans] = answer_weights.get(ans, 0.0) + w
    best_answer = max(answer_weights, key=answer_weights.get)
    return best_answer, answer_weights[best_answer], answer_weights

def semantic_consensus_weighting(embeddings, answers):
    """
    Implements SCW:
    - Compute pairwise cosine similarity between embeddings.
    - Sum similarity scores for each sample.
    - Sum per answer.
    """
    norms = np.linalg.norm(embeddings, axis=1, keepdims=True)
    unit = embeddings / (norms + 1e-8)
    sims = unit @ unit.T  # cosine similarity matrix
    sum_sims = np.sum(sims, axis=1)
    answer_weights = {}
    for ans, s in zip(answers, sum_sims):
        answer_weights[ans] = answer_weights.get(ans, 0.0) + s
    best_answer = max(answer_weights, key=answer_weights.get)
    return best_answer, answer_weights[best_answer], answer_weights

def baseline_majority_vote(answers):
    """Return most frequent answer."""
    from collections import Counter
    counter = Counter(answers)
    return counter.most_common(1)[0][0], counter.most_common(1)[0][1], dict(counter)

def main():
    parser = argparse.ArgumentParser(description="Apply weighting methods.")
    parser.add_argument("--input_file", type=str, default="raw_outputs.json",
                        help="Input JSON from generate.py")
    parser.add_argument("--output_dir", type=str, default="results",
                        help="Directory to write results.")
    parser.add_argument("--model_name", type=str, default="sentence-transformers/all-MiniLM-L6-v2",
                        help="Sentence‑transformer model for embeddings.")
    parser.add_argument("--seed", type=int, default=42,
                        help="Random seed.")
    args = parser.parse_args()

    set_seed(args.seed)
    raw = read_json(args.input_file)
    model = SentenceTransformer(args.model_name, device=0 if torch.cuda.is_available() else -1)

    baseline_res = []
    cpw_res = []
    scw_res = []

    for entry in raw:
        q = entry["question"]
        ground_truth = entry["ground_truth"]
        samples = entry["samples"]
        answers = entry["answers"]

        # Baseline majority vote
        b_ans, b_conf, b_dist = baseline_majority_vote(answers)

        # Embeddings
        emb = compute_embeddings(samples, model)

        # CPW
        cpw_ans, cpw_conf, cpw_dist = centroid_proximity_weighting(emb, answers)

        # SCW
        scw_ans, scw_conf, scw_dist = semantic_consensus_weighting(emb, answers)

        baseline_res.append({
            "question": q,
            "ground_truth": ground_truth,
            "predicted_answer": b_ans,
            "confidence": b_conf,
            "details": {"distribution": b_dist}
        })
        cpw_res.append({
            "question": q,
            "ground_truth": ground_truth,
            "predicted_answer": cpw_ans,
            "confidence": cpw_conf,
            "details": {"distribution": cpw_dist}
        })
        scw_res.append({
            "question": q,
            "ground_truth": ground_truth,
            "predicted_answer": scw_ans,
            "confidence": scw_conf,
            "details": {"distribution": scw_dist}
        })

    output_dir = Path(args.output_dir)
    output_dir.mkdir(parents=True, exist_ok=True)

    write_json(output_dir / "baseline_results.json", baseline_res)
    write_json(output_dir / "cpw_results.json", cpw_res)
    write_json(output_dir / "scw_results.json", scw_res)

    logger.info(f"Results written to {output_dir}")

if __name__ == "__main__":
    import torch
    main()
```
