# Reproduction of "Sample-specific Masks for Visual Reprogramming-based Prompting"

This repository contains the complete implementation to reproduce the results from the paper:
"Sample-specific Masks for Visual Reprogramming-based Prompting" by Chengyi Cai, Zesheng Ye, Lei Feng, Jianzhong Qi, Feng Liu.

## Overview

The paper introduces Sample-specific Multi-channel Masks (SMM), a novel framework for Visual Reprogramming (VR) that replaces the traditional shared mask with sample-specific three-channel masks generated by a lightweight CNN.

The key contributions of this reproduction are:
1. Implementation of the SMM framework with a lightweight mask generator and patch-wise interpolation module
2. Integration with ResNet-18, ResNet-50, and ViT-B32 models
3. Training on 11 benchmark datasets (CIFAR10, CIFAR100, SVHN, GTSRB, Flowers102, DTD, UCF101, Food101, SUN397, EuroSAT, OxfordPets)
4. Reproduction of the results presented in Tables 1, 2, and 3
