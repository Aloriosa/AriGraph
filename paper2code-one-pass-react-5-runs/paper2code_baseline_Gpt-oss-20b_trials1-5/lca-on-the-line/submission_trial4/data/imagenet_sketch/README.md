# Small ImageNet‑Sketch Sample

This folder mirrors the structure of `imagenet_sample`, but contains the corresponding
ImageNet‑Sketch OOD images.  It should contain the same class names, but the images are
sketch‑style renderings.  As with the ID dataset, this is only a tiny subset (~200 images)
used to demonstrate the OOD evaluation.

Place the images under `data/imagenet_sketch/<class_name>/`.
