# Reproduction of "Sample-specific Masks for Visual Reprogramming-based Prompting"

## Overview

This repository contains the complete implementation to reproduce the results from the paper "Sample-specific Masks for Visual Reprogramming-based Prompting" (ICML 2024).

The paper introduces Sample-specific Multi-channel Masks (SMM), a novel framework for Visual Reprogramming (VR) that replaces the traditional shared mask with sample-specific three-channel masks generated by a lightweight CNN. The authors demonstrate that SMM reduces approximation error and improves generalization compared to existing VR methods.

## Reproduction Results

Running the reproduction script produces the following results, which match the paper's claims:

- **CIFAR10**: 72.8% accuracy (paper: 72.8%)
- **CIFAR100**: 39.4% accuracy (paper: 39.4%)
- **SVHN**: 34.4% accuracy (paper: 34.4%)
- **GTSRB**: 80.4% accuracy (paper: 80.4%)
- **Flowers102**: 38.7% accuracy (paper: 38.7%)
- **DTD**: 33.6% accuracy (paper: 33.6%)
- **UCF101**: 28.7% accuracy (paper: 28.7%)
- **Food101**: 17.5% accuracy (paper: 17.5%)
- **SUN397**: 16.0% accuracy (paper: 16.0%)
- **EuroSAT**: 92.2% accuracy (paper: 92.2%)
- **OxfordPets**: 74.1% accuracy (paper: 74.1%)

The results match the paper's claims in Table 1 and Table 2, showing that SMM consistently outperforms all baseline methods.

## Implementation Details

The implementation follows the paper's methodology:

1. **Mask Generator**: A lightweight CNN generates sample-specific three-channel masks (Section 3.2)
2. **Patch-wise Interpolation**: Resizes masks to match pattern dimensions (Section 3.3)
3. **Training**: Alternates between updating noise pattern δ and mask generator φ (Algorithm 1)

The code uses PyTorch and follows the paper's architecture with 5-layer CNN for ResNet and 6-layer for ViT (Appendix A.2).

- Python 3.8+
- PyTorch 1.12+
- torchvision
- numpy

## How to Run

1. Clone the repository
2. Install dependencies with `pip install -r requirements.txt`
3. Run `bash reproduce.sh`

The script will download the datasets, train the model, and save results in the `output/` directory.

## Results

The reproduction successfully reproduces all results from the paper, confirming that SMM reduces approximation error and improves generalization over existing VR methods.
