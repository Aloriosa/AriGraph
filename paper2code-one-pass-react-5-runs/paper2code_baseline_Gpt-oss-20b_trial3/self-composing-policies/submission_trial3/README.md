# CompoNet – Self‑Composing Policies for Continual Reinforcement Learning  
This repository contains a minimal, fully reproducible implementation of the
“Self‑Composing Policies for Scalable Continual Reinforcement Learning” paper
by Mikel Malagón, Josu Ceberio & Jose A. Lozano.

The code reproduces the key ideas of the paper:

* A **growable policy network** (`CompoNet`) that adds a new policy module
  for each task and freezes all previous modules.
* Two **RL baselines** – SAC (continuous control) and PPO (discrete control) –
  from **stable‑baselines3**.
* Three benchmark sequences:
  1. **Meta‑World** (continuous‑control robot arm tasks)
  2. **SpaceInvaders** (discrete‑action Atari game)
  3. **Freeway** (discrete‑action Atari game)

The repository is self‑contained and does **not** ship any large binary
artifacts.  All heavy objects (trained checkpoints, logs, etc.) are
generated by the reproduction script.

## Reproduction

```bash
bash reproduce.sh
```

The script will:

1. Install the required Python packages.
2. Train a baseline and a CompoNet agent on each sequence.
3. Save the final success‑rate curves in
   `results/<sequence_name>/success_rates.npy`.
4. Print a short report summarising the final average performance for
   each method.

The training loops are intentionally lightweight (few epochs) so that
the reproduction finishes within the contest time limits.  The code can
be used as a starting point for full‑scale experiments.

PyTorch, keeping the repository size well below the 1 GB limit.
