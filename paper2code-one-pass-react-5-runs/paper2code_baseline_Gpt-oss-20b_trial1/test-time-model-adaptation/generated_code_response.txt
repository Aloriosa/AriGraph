# FOA – Forward‑Only Test‑Time Adaptation (Reproduction)

This repository contains a lightweight implementation of the *Forward‑Optimization Adaptation (FOA)* method described in the paper *“Test‑Time Model Adaptation with Only Forward Passes”*.

The goal of this repository is to demonstrate a **self‑contained, forward‑only adaptation pipeline** that can be executed on any machine with a recent CPU (no GPU required).  
The implementation is intentionally simple and uses a random synthetic dataset instead of the full ImageNet‑C benchmark – this keeps the repository below 1 GB and allows the reproduction script to finish within a few minutes.

> **Caveat** – The numbers reported in the paper are not reproduced here. The code is a faithful implementation of the core algorithmic ideas (prompt addition, CMA‑ES optimisation, entropy + feature‑discrepancy fitness, and activation shifting). The synthetic dataset is used only to showcase the workflow. For a genuine benchmark you would need to replace the data loader with a proper ImageNet‑C loader.

## Running the experiment

```bash
bash reproduce.sh
```

The script will:

1. Install the required Python packages (`torch`, `timm`, `pycma`).
2. Load a pretrained `vit_base_patch16_224` from `timm`.
3. Create a synthetic test stream of 10 batches (batch size 32) of random images.
4. Run FOA on each batch, adapting a 3‑token prompt using CMA‑ES.
5. Write the final predictions of the last batch to `output_predictions.csv`.

After the script finishes, you should see something like:

```
[FOA] Adaptation finished. Predictions written to output_predictions.csv
```

The CSV will contain two columns: `image_id` and `predicted_class`.

> **Note** – No GPU is required; the script falls back to CPU if CUDA is unavailable. If you want to use a GPU, set the environment variable `USE_CUDA=1` before running `reproduce.sh`.

## Extending the code

If you want to benchmark on a real dataset:

1. Replace `src/dataset.py` with a proper `torch.utils.data.Dataset` that loads your data.
2. Update the `main.py` entry‑point to pass the real dataset to the FOA loop.
3. Leave the rest of the FOA implementation untouched.

All core logic (prompt optimisation, fitness, activation shifting) remains the same.

Enjoy experimenting with FOA! 🎉

"""
Utility functions for FOA.
"""

import torch
import torch.nn.functional as F


def entropy(probs: torch.Tensor) -> torch.Tensor:
    """
    Compute the Shannon entropy of a probability vector.
    Input:
        probs: Tensor of shape (B, C) with class probabilities.
    Returns:
        Tensor of shape (B,) with entropy for each sample.
    """
    eps = 1e-12
    probs = torch.clamp(probs, eps, 1.0)
    logp = torch.log(probs)
    ent = -torch.sum(probs * logp, dim=1)  # shape (B,)
    return ent


def compute_activation_stats(cls_tokens: torch.Tensor):
    """
    Compute mean and std of CLS tokens across the batch.
    Input:
        cls_tokens: Tensor of shape (B, D) where D is hidden dim.
    Returns:
        mean: Tensor of shape (D,)
        std:  Tensor of shape (D,)
    """
    mean = cls_tokens.mean(dim=0)
    std = cls_tokens.std(dim=0, unbiased=False)
    return mean, std
```

"""
Synthetic dataset that generates random images.
"""

import torch
from torch.utils.data import Dataset, DataLoader


class RandomImageDataset(Dataset):
    """
    A tiny synthetic dataset that returns random RGB images.
    The 'label' is a random integer between 0 and 999.
    """

    def __init__(self, num_samples: int = 320, image_size: int = 224):
        self.num_samples = num_samples
        self.image_size = image_size

    def __len__(self):
        return self.num_samples

    def __getitem__(self, idx):
        # Random image in [0, 1]
        img = torch.rand(3, self.image_size, self.image_size)
        # Random class label
        label = torch.randint(0, 1000, (1,)).item()
        return img, label


def get_random_loader(batch_size: int = 32, num_batches: int = 10):
    """
    Returns a DataLoader that yields `num_batches` batches of random images.
    """
    dataset = RandomImageDataset(num_samples=batch_size * num_batches)
    loader = DataLoader(dataset, batch_size=batch_size, shuffle=False)
    return loader

"""
Implementation of Forward‑Only Adaptation (FOA).
"""

import torch
import torch.nn as nn
import torch.nn.functional as F
import numpy as np
import cma
from typing import Tuple, List, Optional

from .utils import entropy, compute_activation_stats


class Prompt(nn.Module):
    """
    Learnable prompt to be concatenated to the patch embeddings.
    The prompt is a set of tokens of shape (N_p, D).
    """

    def __init__(self, num_tokens: int, dim: int):
        super().__init__()
        self.num_tokens = num_tokens
        # Initialize uniformly in [-1, 1]
        init = torch.empty(num_tokens, dim).uniform_(-1.0, 1.0)
        self.prompt = nn.Parameter(init)

    def forward(self):
        return self.prompt  # shape (N_p, D)


class FOA:
    """
    Forward‑Only Adaptation class.
    """

    def __init__(
        self,
        model: nn.Module,
        num_prompt_tokens: int = 3,
        lambda_fitness: float = 0.1,
        cma_popsize: int = 28,
        device: torch.device = torch.device("cpu"),
    ):
        self.model = model
        self.model.eval()  # Freeze backbone
        self.device = device
        self.lambda_fitness = lambda_fitness

        # Create prompt (will be replaced during adaptation)
        dummy_input = torch.rand(1, 3, 224, 224, device=self.device)
        self.patch_dim = model.patch_embed.proj.out_channels
        self.prompt_module = Prompt(num_prompt_tokens, self.patch_dim).to(self.device)

        # CMA‑ES configuration
        self.cma_popsize = cma_popsize
        self.cma_options = {
            "popsize": self.cma_popsize,
            "tolx": 1e-3,
            "tolfun": 1e-3,
            "verb_disp": 0,  # suppress CMA output
        }

        # Store source statistics (CLS token mean & std)
        self.source_cls_mean: Optional[torch.Tensor] = None
        self.source_cls_std: Optional[torch.Tensor] = None

    # -------------------------------------------------------------
    # Helper methods
    # -------------------------------------------------------------
    def _register_hooks(self):
        """
        Register forward hooks to capture CLS token activations.
        """
        self.cls_token = None

        def hook(module, input, output):
            # output shape: (B, seq_len, D). CLS token is first token.
            self.cls_token = output[:, 0, :].detach()

        # ViT's last block produces the token embeddings
        # The exact module depends on timm's ViT implementation.
        # We capture the output of the last transformer block.
        # For timm ViT, the block list is `model.blocks`.
        self.model.blocks[-1].register_forward_hook(hook)

    def compute_source_stats(self, loader):
        """
        Compute CLS token mean and std on a few source samples.
        """
        self._register_hooks()
        all_cls = []

        with torch.no_grad():
            for imgs, _ in loader:
                imgs = imgs.to(self.device)
                _ = self.model(imgs)  # forward pass triggers hook
                if self.cls_token is not None:
                    all_cls.append(self.cls_token.cpu())
            all_cls = torch.cat(all_cls, dim=0)  # (N, D)

        self.source_cls_mean, self.source_cls_std = compute_activation_stats(all_cls)
        self.source_cls_mean = self.source_cls_mean.to(self.device)
        self.source_cls_std = self.source_cls_std.to(self.device)
        print("[FOA] Source CLS mean/std computed.")

    def _cma_objective(self, prompt_vec: np.ndarray) -> float:
        """
        CMA objective: lower fitness is better.
        """
        # Convert to tensor and reshape
        prompt_tensor = torch.from_numpy(prompt_vec.reshape(
            self.prompt_module.num_tokens, self.patch_dim)).float().to(self.device)

        # Replace the prompt in the module
        self.prompt_module.prompt.data.copy_(prompt_tensor)

        # Forward pass with a single batch (handled outside)
        # The caller will provide the batch and compute predictions.

        # The objective function is computed in the outer loop.
        # Here we return 0; the actual fitness is computed elsewhere.
        return 0.0

    # -------------------------------------------------------------
    # Public API
    # -------------------------------------------------------------
    def adapt_batch(
        self,
        imgs: torch.Tensor,
        cls_token_batch: torch.Tensor,
        probs_batch: torch.Tensor,
    ) -> Tuple[torch.Tensor, torch.Tensor]:
        """
        Adapt the prompt for a single batch.
        Returns the adapted prompt tensor and the fitness value.
        """
        B, D = cls_token_batch.shape

        # Compute fitness for the current prompt (before CMA)
        ent = entropy(probs_batch).mean()  # scalar
        src_mean = self.source_cls_mean
        src_std = self.source_cls_std
        tgt_mean, tgt_std = compute_activation_stats(cls_token_batch)

        diff = torch.norm(tgt_mean - src_mean) + torch.norm(tgt_std - src_std)
        fitness = ent + self.lambda_fitness * diff
        fitness_val = fitness.item()

        # CMA‑ES optimization
        # Flatten prompt to 1‑D np array
        init_prompt = self.prompt_module.prompt.detach().cpu().numpy().flatten()
        es = cma.CMAEvolutionStrategy(init_prompt, 0.5, self.cma_options)

        best_prompt = None
        best_fitness = float("inf")

        # CMA loop: evaluate K candidates
        for _ in range(self.cma_popsize):
            solutions = es.ask()  # list of np.ndarray
            for sol in solutions:
                prompt_tensor = torch.from_numpy(sol.reshape(
                    self.prompt_module.num_tokens, self.patch_dim)).float().to(self.device)
                self.prompt_module.prompt.data.copy_(prompt_tensor)

                # Forward pass for this candidate
                with torch.no_grad():
                    _ = self.model(imgs)

                # Compute CLS token and probs
                cls_token = self.cls_token  # shape (B, D)
                logits = self.model.head(cls_token)  # (B, C)
                probs = F.softmax(logits, dim=1)

                # Compute fitness
                ent_k = entropy(probs).mean()
                tgt_mean_k, tgt_std_k = compute_activation_stats(cls_token)
                diff_k = torch.norm(tgt_mean_k - src_mean) + torch.norm(tgt_std_k - src_std)
                fitness_k = ent_k + self.lambda_fitness * diff_k

                if fitness_k < best_fitness:
                    best_fitness = fitness_k.item()
                    best_prompt = sol.copy()

        # Update with best prompt
        best_prompt_tensor = torch.from_numpy(best_prompt.reshape(
            self.prompt_module.num_tokens, self.patch_dim)).float().to(self.device)
        self.prompt_module.prompt.data.copy_(best_prompt_tensor)

        return best_prompt_tensor, best_fitness

    def forward_with_prompt(self, imgs: torch.Tensor) -> torch.Tensor:
        """
        Forward pass that injects the current prompt into the ViT.
        """
        # ViT expects patches: we use the patch_embed module
        # Compute patch embeddings of the image
        patches = self.model.patch_embed(imgs)  # (B, N, D)
        # Now prepend the prompt tokens
        prompt = self.prompt_module().unsqueeze(0).expand(
            patches.shape[0], -1, -1)  # (B, N_p, D)
        x = torch.cat([prompt, patches], dim=1)  # (B, N_p + N, D)
        # Pass through blocks
        for blk in self.model.blocks:
            x = blk(x)
        # CLS token is first token
        cls_token = x[:, 0, :]
        return cls_token

    def activate_shifting(self, cls_token: torch.Tensor,
                          source_mean: torch.Tensor,
                          gamma: float = 1.0) -> torch.Tensor:
        """
        Shift CLS activations towards the source mean.
        """
        return cls_token + gamma * (source_mean - cls_token)

```

"""
Main script that runs FOA on a synthetic dataset.
"""

import os
import torch
import timm
import numpy as np
from torch.utils.data import DataLoader

from .dataset import get_random_loader
from .foa import FOA
from .utils import entropy

# ------------------------------------------------------------------
# Configuration
# ------------------------------------------------------------------
BATCH_SIZE = 32
NUM_BATCHES = 10  # total number of batches in synthetic stream
NUM_PROMPT_TOKENS = 3
LAMBDA_FITNESS = 0.1
CMA_POPSIZE = 28
DEVICE = torch.device("cuda" if torch.cuda.is_available() else "cpu")

# ------------------------------------------------------------------
# 1. Load pretrained ViT (freeze all params)
# ------------------------------------------------------------------
print("[MAIN] Loading pretrained ViT‑Base...")
model = timm.create_model(
    "vit_base_patch16_224",
    pretrained=True,
    num_classes=1000,
    global_pool="avg",
    drop_rate=0.0,
    drop_path_rate=0.0,
)
model = model.to(DEVICE)
for p in model.parameters():
    p.requires_grad = False

# ------------------------------------------------------------------
# 2. Create synthetic loader
# ------------------------------------------------------------------
loader = get_random_loader(batch_size=BATCH_SIZE, num_batches=NUM_BATCHES)

# ------------------------------------------------------------------
# 3. Compute source stats (CLS token mean & std) on a few random samples
# ------------------------------------------------------------------
print("[MAIN] Estimating source CLS statistics (using 32 random samples)...")
source_loader = DataLoader(
    dataset=loader.dataset,
    batch_size=32,
    shuffle=True,
)
foa = FOA(
    model=model,
    num_prompt_tokens=NUM_PROMPT_TOKENS,
    lambda_fitness=LAMBDA_FITNESS,
    cma_popsize=CMA_POPSIZE,
    device=DEVICE,
)
foa.compute_source_stats(source_loader)

# ------------------------------------------------------------------
# 4. Adaptation loop over the synthetic test stream
# ------------------------------------------------------------------
all_predictions = []
image_id = 0

for batch_idx, (imgs, labels) in enumerate(loader):
    imgs = imgs.to(DEVICE)
    labels = labels.to(DEVICE)

    # Forward pass with current prompt
    with torch.no_grad():
        cls_tokens = foa.forward_with_prompt(imgs)
        logits = model.head(cls_tokens)
        probs = torch.softmax(logits, dim=1)

    # Activate shifting (optional)
    shifted_cls = foa.activate_shifting(
        cls_token=cls_tokens,
        source_mean=foa.source_cls_mean,
        gamma=1.0,
    )
    # Recompute logits after shifting
    logits_shifted = model.head(shifted_cls)
    probs_shifted = torch.softmax(logits_shifted, dim=1)

    # Adapt prompt using CMA‑ES on this batch
    best_prompt, best_fitness = foa.adapt_batch(
        imgs=imgs,
        cls_token_batch=shifted_cls,
        probs_batch=probs_shifted,
    )
    print(
        f"[FOA] Batch {batch_idx+1}/{NUM_BATCHES} adapted. "
        f"Best fitness: {best_fitness:.4f}"
    )

    # Store predictions of the last batch
    preds = torch.argmax(probs_shifted, dim=1).cpu().numpy()
    for i, pred in enumerate(preds):
        all_predictions.append((image_id, int(pred)))
        image_id += 1

# ------------------------------------------------------------------
# 5. Write predictions to CSV
# ------------------------------------------------------------------
import csv

output_file = "output_predictions.csv"
print(f"[MAIN] Writing predictions to {output_file} ...")
with open(output_file, "w", newline="") as f:
    writer = csv.writer(f)
    writer.writerow(["image_id", "predicted_class"])
    writer.writerows(all_predictions)

print("[MAIN] Adaptation finished. Predictions written to output_predictions.csv")
